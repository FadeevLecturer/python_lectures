Практические задания#

Повторить redemo.py#

Вместе с стандартным дистрибутивом python поставляются демонстрационные примеры. В частности скрипт redemo.py, который реализует приложения для тестирования регулярных выражений на основе tkinter. Найти этот файл можно 1) по пути python_installation_folder/Tools/demo/redemo.py, где python_installation_folder — путь к папке с установленным python; 2) в репозитории с примерами по PySide — файл redemo.py.

В качестве практического упражнения предлагается повторить функционал этого приложения, но используя PySide для реализации графического интерфейса.

Папка с изображениями#

Реализуйте программу, которая позволит в интерфейсе выбирать папку с изображениями и просматривать их в порядке создания, листая их кнопками на клавиатуре (например, стрелками).
